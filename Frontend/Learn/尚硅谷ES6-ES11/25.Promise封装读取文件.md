# 这节课的重要知识点是node的开发环境API的搭建

#Script #教程

1. 首先需要在我的百度网盘中下载支持IE7版本的node 13.4.0压缩包然后解压到D盘根目录之下

2. 右键我的电脑-->属性-->高级系统设置-->环境变量-->系统变量-->Path-->编辑,多个路径之间用;分号隔开 ;D:\node-v13.4.0-win-x64 粘贴连点两下确定就行了


3. 打开powershell,输入node -v检查是否安装成功,如果出现版本号则安装成功,输入npm -v检查是否安装成功

4. 如果报错:无法将"node"项识别为cmdlet、 函数、脚本文件或可运行程序的名称。

- 重启电脑
